📋 SOBRE O PROJETO:

O Muscle Levels é uma plataforma web completa para criação e acompanhamento 
de treinos físicos, desenvolvida com Next.js 15, React 19, TypeScript, 
Tailwind CSS e PostgreSQL.

PRINCIPAIS FUNCIONALIDADES:
✅ Sistema de autenticação (Email + Google OAuth)
✅ Gerador inteligente de treinos personalizados
✅ 200+ exercícios catalogados com GIFs
✅ Editor drag & drop de exercícios
✅ Estatísticas e gráficos de progressão
✅ Sistema de gamificação e ranking
✅ Suporte a 6 idiomas
✅ Design responsivo (mobile/tablet/desktop)
✅ PWA instalável

════════════════════════════════════════════════════════════════════════

💻 TECNOLOGIAS:

• Next.js 15 + React 19 + TypeScript
• Tailwind CSS + DaisyUI + Radix UI
• Prisma ORM + PostgreSQL
